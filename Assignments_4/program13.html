13) Shopping Cart Discounts (Nested If-Else) 
    Problem:  
    Discount rules: 
    If cart value > ₹10,000 → 20% discount 
    If cart value 5000–9999 → 10% discount 
    If cart value < 5000 but customer is "Premium" → 5% discount 
    Otherwise → no discount 
    But, if today is "Black Friday", apply flat 50% discount regardless of other rules. Implement 
    using nested if-else ladder. 
    Sample Test Cases:  
    (12000,‘Normal’,‘Friday’) → 20%  
    (4000,‘Premium’,‘Monday’) → 5%  
    (4000,‘Normal’,‘Monday’) → 0%  
    (3000,‘Normal’,‘Black Friday’) → 50%
