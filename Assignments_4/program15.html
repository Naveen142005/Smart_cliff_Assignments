15) Airline Check-in (Switch + Nested If) 
Problem:  
Passenger rules: 
If class = "Business" → Priority Check-in 
If class = "Economy" → Normal Check-in 
If baggage > 30kg → "Excess Baggage Fee" 
If passenger is "VIP" → Always "Priority Check-in" regardless of class 
Use switch-case combined with if conditions to handle all cases. 
VIP → Priority regardless. Else Business→Priority; Economy→Normal. If baggage>30 → add 
“Excess Baggage Fee”. 
Sample Test Cases:  
(VIP, Economy, 25) → “Priority Check-in”  
(Business, 35) → “Priority Check-in + Excess Baggage Fee”  
(Economy, 20) → “Normal Check-in”
