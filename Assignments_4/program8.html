8) Smart Elevator (Switch + Nested If) 
Problem:  
An elevator system handles requests: 
Floor 0 = Lobby → "Welcome" 
Floor 1–5 = Offices → "Go to Office" 
Floor 6–9 = Cafeteria → "Go to Cafeteria" 
Floor 10 = Roof → "Enjoy the view" 
Any other → "Invalid floor" 
Additionally, if the elevator is in maintenance mode, no request should be served regardless 
of floor.  
Sample Test Cases:  
(floor=0, maint=false) → “Welcome”  
(floor=7, maint=false) → “Go to Cafeteria”  
(floor=7, maint=true) → “Maintenance” - (floor=100, maint=false) → “Invalid floor”
