9) Online Exam Rules  
Problem:  
A student can submit the exam only if: 
Time left > 0, AND 
Student has answered at least 1 question, AND 
Has not already submitted. 
If any condition fails, output "Submission Failed" with the reason. Implement using if-else 
with compound conditions. 
Sample Test Cases:  
time=10, answered=5, submitted=false → “Submission Accepted”  
time=0, answered=5, submitted=false → “Submission Failed: No time left”  
time=10, answered=0, submitted=false → “Submission Failed: No answers”  
time=10, answered=2, submitted=true → “Submission Failed: Already submitted”
