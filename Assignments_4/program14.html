14 Student Result Processing (for…in + Conditions) 
    Problem:  
    An object stores student marks: 
    Rules: 
    Any subject 35 = "Fail" 
    If avg ≥ 75 and no fails → "Distinction" 
    If avg 50–74 and no fails → "Pass" 
    Otherwise → "Fail" 
    Use for...in loop with conditions. 
    Sample Test Cases:  
    {math:90, science:25, english:80, history:34, computer:99} → Fail (due to <35)  
    {math:80, science:75, english:78} → Distinction  
    {math:60, science:55, english:50} → Pass
